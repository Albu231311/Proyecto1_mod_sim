# Proyecto TSP: Comparación GA vs ILP

Este proyecto implementa y compara dos aproximaciones para resolver el Traveling Salesman Problem (TSP):
1. Un algoritmo genético (GA) con optimización 2-opt
2. Una formulación de programación lineal entera (ILP) usando el modelo MTZ

## Requisitos
- Python
- Dependencias listadas en `requirements.txt`

Para instalar las dependencias:
```bash
pip install -r requirements.txt
```
